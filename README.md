# Titanic Survival Prediction 🚢

## 📌 Project Overview
The Titanic Survival Prediction project aims to predict whether a passenger survived the Titanic disaster based on features such as age, gender, class, and fare. This project applies data analysis, feature engineering, and machine learning algorithms to create a predictive model.

## 🎯 Objectives
- Analyze the Titanic dataset and understand the key factors affecting survival.  
- Build a machine learning model to predict survival outcomes.  
- Evaluate model performance using appropriate metrics.  
- Gain hands-on experience in data preprocessing, visualization, and predictive modeling.

## 📂 Dataset
The dataset used in this project is the **Titanic dataset** from [Kaggle](https://www.kaggle.com/c/titanic).  
It contains information about:
- PassengerId  
- Survived (target variable)  
- Pclass (Passenger class)  
- Name, Sex, Age  
- SibSp (siblings/spouses aboard), Parch (parents/children aboard)  
- Ticket, Fare, Cabin, Embarked  

## ⚙️ Technologies Used
- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn  
- **IDE:** Jupyter Notebook / VS Code  
- **Version Control:** Git & GitHub  

## 🧠 Machine Learning Algorithms
- Logistic Regression  
- Decision Tree Classifier  
- Random Forest Classifier  
- Support Vector Machine (SVM)  
- Gradient Boosting (optional)  

## 🔄 Project Workflow
1. Data Collection & Loading  
2. Data Cleaning & Preprocessing (handling missing values, encoding categorical features)  
3. Exploratory Data Analysis (EDA)  
4. Feature Engineering  
5. Model Selection & Training  
6. Model Evaluation (Accuracy, Precision, Recall, F1-Score)  
7. Predictions on Test Data  
8. Visualization of Results  

## 📊 Results
- Achieved high accuracy in predicting survival using ensemble methods.  
- Key factors influencing survival: **Gender**, **Passenger Class**, **Age**, **Fare**.  
- Confusion matrix and classification report indicate the effectiveness of the model.
